GIT  COMMANDS

process to setup GIT HUB initially
-------------------------------------
git config --global user.name "aaaaaaaaaa"  (Tell Git your name so your commits will be properly labeled.)


git config --global user.email "aaaaaa@gmail.com"


git init (initialize repository)




// associate remote repo to local repo

git remote add origin https://github.com/githubrepo938/scripts.git

git pull https://github.com/githubrepo938/scripts.git

git push -u origin master
